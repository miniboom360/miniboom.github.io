钓鱼演练的流程

软硬件准备(gophish+邮件服务器+云主机) => 绕过安全防护 => 钓鱼文本 => 钓鱼网站搭建 => 钓鱼邮件附件 => 进行数据展示。

1、gophish+邮件服务器，gophish用于伪造钓鱼网站和统计点击钓鱼链接，点击钓鱼邮件附件等数据；邮件服务器用于批量发送邮件。

2、判断是否有邮件网关服务器、SPF、DKIM、DMARC安全防御策略，如果有，需要通过技术手段进行绕过，不然会被投入到垃圾邮件和拒收。

3、钓鱼文本内容，准备文案，获取受害者的信任和促使他们点击。

4、钓鱼网站展示，准备一个和谎称正常页面高度相似的网站，必要时配备高仿度域名以迷惑受害者

5、钓鱼邮件附件，准备一个过免杀的恶意exe（绕过360，火绒安全，卡巴斯基等杀毒软件的封锁），让受害者点击后可以被记录到机器和用户名，用于统计次数哪些用户，点击了恶意exe

6、通过gophish进行展示，有哪些用户是点击了钓鱼链接，哪些访问了钓鱼网站并留下了信息，哪些是点击了附件。

我们这边需要部署钓鱼平台，需要解开25端口限制，以批量发送邮件
